# google_file_search_api_testing

To test the file search api

-- Clone the repository
-- Add a Virtual Environment then Activate it and Install all the requiremnets
-- Add the document you want to add to the RAG system (file search api tool) in the data folder.
-- Add an .env file and intialize your Google Api Token or directly paste your API token in the file_search_test.py file for testing.
-- In the file_search_test.py file configure the location of your document and add your query under contents then run the file.
-- The test.py file is used to check or display the list of files you have uploaded to the google file search api.
